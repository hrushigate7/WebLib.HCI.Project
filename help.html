RENEW & RETURN

Here is what you need to know:


RENEW

  • If your book period has expired and want to RENEW the book, your book period

will be automatically renewed for 20 more days.


RETURN

  • You can return circulating library materials to any PCCOE library.


FINE COLLECTION


  • Fees for lost or long overdue materials vary depending on the item
  • See below section for more details

If you still have questions about library fines, contact the PCCOE Librarian

Fine Estimations


  • Fine for late return of tech books: Rs.1.00 per day
  • Fine for late return of non-tech books: Rs.0.50 per day
  • Fine for improper maintainance of books: Rs. 50 per book
